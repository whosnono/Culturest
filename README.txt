1. Tener instalado ruby y sinatra
- descargar ruby desde este link: https://rubyinstaller.org/downloads/
- instalar sinatra desde la terminarl: gem install sinatra

2. En terminal navergar hasta la ruta donde se encuentra el archivo exportador.rb , y ejecutar el siguiente comando: 
- ruby exportador.rb

3. Abre tu navegador y ve a http://localhost:4567.
